LTI System: A Linear Time-Invariant (LTI) system's output depends linearly on the input and doesn't change over time. For example, a resistor-capacitor (RC) circuit is an LTI system, where the output voltage depends on the input voltage. Open-Loop System: An open-loop system operates without feedback.
